A template for working with purescript and halogen, and using webapck as bundler.

Installing dependencies:
```
npm install
```
or (yarn):
```
yarn install
```

Building project for production:
```
npm run build
```
or (yarn):
```
yarn build
```

Launching interactive server:
```
npm run serve
```
or (yarn):
```
yarn serve
```

Running unit tests:
```
npm run test
```
or (yarn):
```
yarn test
```
